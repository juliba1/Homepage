# Homepage

## Description

**Link Homepage** is a simple HTML web project that serves as a central hub for all important links.
The page is divided into various categories, such as **HTWG**, **Streaming**, **Coding**, and **Shopping**, with each section displaying links as buttons in an organized manner.
This project helps to quickly access the most frequently used online resources.

## Features

- **Clear Structure**: The homepage is divided into several categories to facilitate navigation.
- **Link Buttons**: Each link is displayed as a button to enhance the user experience and make the links visually appealing.
- **Responsive Design**: The page adapts to different screen sizes and is easily readable on both desktop and mobile devices.

## Technologies

- **HTML**: Structure of the webpage
- **CSS**: Styling and layout of the page
